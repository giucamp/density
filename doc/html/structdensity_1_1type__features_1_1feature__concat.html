Detailed Description

template<typename FIRST, typename SECOND>
struct density::type_features::feature_concat< FIRST, SECOND >

This template concatenates two feature_list, or a feature_list and a feature.

Template Parameters
FIRSTfeature_list to prepend
SECONDfeature_list or feature to append

The inner member type is an alias for the concatenations of all features in the tempate arguments. The order of the features is preserved.

The alias feature_concat_t can be used instead of feature_concat<...>::type:

template <typename FIRST, typename SECOND>
using feature_concat_t = typename feature_concat<FIRST, SECOND>::type;

Example:

using namespace type_features;
using MyPartialFeatures = feature_list<default_construct, size>;
using MyFeatures = feature_concat_t<MyPartialFeatures, alignment>;
using MyFeatures1 = feature_concat_t<MyPartialFeatures, feature_list<alignment> >;
static_assert(std::is_same<MyFeatures, MyFeatures1>::value, "");
